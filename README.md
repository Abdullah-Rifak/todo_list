# TODO App

A simple full-stack TODO application with a React frontend and an Express + MongoDB backend.

## Project Layout

- [backend/README.md](backend/README.md) covers the API, environment variables, and available endpoints.
- [frontend/README.md](frontend/README.md) covers the React app setup and client behavior.
- [assesment.md](assesment.md) contains the original assessment notes for the project.

## Quick Start

1. Set up the backend first so the API is available.
2. Start the frontend after the backend is running.
3. Follow the per-app README files for the exact commands and environment variables.

## What This Repo Contains

- `backend/` - Express + MongoDB API for managing todos
- `frontend/` - React UI for creating, editing, completing, and deleting todos

## Stack

- Node.js
- Express
- MongoDB
- React
